Software QA Engineer

Job Description

    Sreemaata technologies is looking for a Self Motivated Software QA Engineer for the Dublin, OH location with the following responsibilities.
  • Review requirement specifications, designs and develop test plans based on functional and design specifications.
  • Create test strategy, test measurement metrics, performance tests, unit tests, regression tests, usability and security tests.
  • Provide feedback to Software development team on design and quality aspects.
  • Perform validation across multiple environments including onprem, cloud and hybrid deployments.
  • Use collaboration and issue tracking tools like JIRA to create test metrics, dashboard and reports.
  • Provide estimates for software test work units
  • Use release, build and deploy tools to deploy software across multiple environments.
  • Develop complex SQL and Big data queries to perform end to end data validations.

Requirements

  • Bachelors Degree in Computer Engineering, Computer Science, Systems Engineering or a related technical field.
  • Preferred Qualification : MS Degree in Computer Engineering, Computer Science, Systems Engineering or a related technical field
  • 1 - 3 Years of Software Testing experience
